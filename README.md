# 魔法跳跃 (Magic Jump)

一个功能丰富的HTML5平台跳跃游戏，包含成长系统、BOSS战、技能商店和多种道具。

## 游戏特色

### 🎮 核心玩法
- **3个关卡**：魔法森林、炽热沙漠、冰雪王国
- **5种属性成长**：跳跃强化、疾风步、护盾精通、能量恢复、财富之手
- **BOSS战斗**：每个关卡都有独特的BOSS，需要技巧击败
- **成长系统**：经验值、等级提升、属性点分配

### 🛠️ 技能系统
- **二段跳**：在空中进行第二次跳跃
- **护盾**：抵挡一次伤害
- **冲刺**：快速向前冲刺，可攻击BOSS
- **技能能量**：技能需要能量，随时间恢复

### ⚡ 道具系统
- **磁铁**：自动吸引星星
- **加速**：增加移动速度
- **倍数**：分数加倍
- **无敌药水**：短暂无敌状态
- **时间减速**：减慢游戏速度
- **跳跃靴子**：增强跳跃能力

### 🎵 音效系统
- 8-bit风格背景音乐，每个关卡不同音调
- 完整的音效系统（跳跃、收集、攻击、死亡等）
- 可开关的音乐和音效

## 控制方式

| 按键 | 功能 |
|------|------|
| ← → | 左右移动 |
| 空格/↑ | 跳跃 |
| Z | 二段跳 |
| X | 激活护盾 |
| C | 冲刺 |
| M | 切换音乐 |
| N | 切换音效 |

## BOSS战斗机制

每个BOSS有不同的攻击模式：
- **古树精**：藤蔓鞭打、孢子云、根刺
- **沙虫王**：钻地突袭、沙暴、尾巴横扫
- **冰霜巨人**：冰霜射线、地面重击、寒冰新星

**攻击方式**：
1. **踩踏攻击**：跳到BOSS上方下落时造成25点伤害
2. **冲刺攻击**：激活冲刺技能时撞到BOSS造成15点伤害

   ```bash
   python -m http.server 8080
   ```

2. 打开浏览器访问：
   ```
   http://localhost:8080
   ```

## 技术特点

- 纯HTML5/CSS3/JavaScript实现，无外部依赖
- Canvas 2D渲染，支持粒子特效
- 完整的游戏状态管理
- 本地存储保存进度
- 响应式设计，支持多种屏幕
